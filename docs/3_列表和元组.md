# 列表和元组

## 通用序列

### 索引

0 开始，-1 结束

### 分片

```python
a[1:2] #索引1-2，不包含最后的2
a[-3:-1] #倒数第三和倒数第二
a[-3:] #倒数第三到最后
a[:3] #前三个
a[1:10:2] #step2 步伐是2
#step可以正整数，负整数
```

### 序列相加

列表相加，得到列表拼接。

### 乘法

字符串、列表可以使用乘法，重复

### 成员资格

in 表示是否为真，返回的值是布尔运算符。True, False

### 长度

`len(a)`

### 最大值 最小值

`max([1,2,3])` 和 `min([1,2,3])`
